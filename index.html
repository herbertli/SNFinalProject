Visualizing the Senate

Introduction
Social Networks Spring '17 Final Project

In this project I'll try to see whether the Senate can be modeled as a social network. Members of the Senate will be connected with each other based on how "similar" they are. In this project, I'll try two different measures of similarity to build the network:

  1. A network based on voting trends between senators
  2. A network based on donations each senator received from a specific industry
Thanks to OpenSecrets.org and the Center for Responsive Politics for the campaign finance data.
And thanks to Govtrack.us for the historical voting data.

Voting Network
Senators A is connected with Senator B if they vote the same way x% of the time

For this similarity measure, I used cosine similarity, bounded in [0, 1]. We count the number of times Senator A votes the same way as Senator B and normalize the result so it is independent of the number of votes either senator participated in. As a detail of the implementation, if both senators abstain for a particular, it does not count as them voting the same way. Feel free to move the slider in order to adjust the threshold.

Money Network
Senators A is connected with Senator B if they received donations from x% of the same industries

For this similarity measure, I again used cosine similarity, bounded in [0, 1]. We count the number of industries Senator A shares with Senator B and normalize the result so it is independent of the total number of industries that donated to either senator. Furthermore, we remove any lower outliers from each senator because we only really want to look at significant donations. Feel free to move the slider in order to adjust the threshold.
